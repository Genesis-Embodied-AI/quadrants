   Alias for :class:`~quadrants.types.ndarray_type.NdarrayType`.

   Example::

       >>> @qd.kernel
       >>> def to_numpy(x: qd.types.ndarray(), y: qd.types.ndarray()):
       >>>     for i in range(n):
       >>>         x[i] = y[i]
       >>>
       >>> y = qd.ndarray(qd.f64, shape=n)
       >>> ... # calculate y
       >>> x = numpy.zeros(n)
       >>> to_numpy(x, y)  # `x` will be filled with `y`'s data.

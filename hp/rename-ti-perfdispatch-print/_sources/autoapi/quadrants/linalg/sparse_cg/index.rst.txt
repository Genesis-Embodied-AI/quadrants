.. py:class:: SparseCG(A, b, x0=None, max_iter=50, atol=1e-06)

   Conjugate-gradient solver built for SparseMatrix.

   Use conjugate-gradient method to solve the linear system Ax = b, where A is SparseMatrix.

   :param A: The coefficient matrix A of the linear system.
   :type A: SparseMatrix
   :param b: The right-hand side of the linear system.
   :type b: numpy ndarray, quadrants Ndarray
   :param x0: The initial guess for the solution.
   :type x0: numpy ndarray, quadrants Ndarray
   :param max_iter: Maximum number of iterations.
   :type max_iter: int
   :param atol: Tolerance(absolute) for convergence.
